# 🚀 Run Locally

### 1️⃣ Clone the repository

```
git clone https://github.com/nishant150226-hub/forest-game.git
```

### 2️⃣ Enter the project folder

```
cd forest-game
```

### 3️⃣ Start a local server

Example using Python:

```
python -m http.server
```

### 4️⃣ Open in browser

```
http://localhost:8000
```
